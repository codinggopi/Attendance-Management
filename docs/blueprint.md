# **App Name**: A+ Attendance

## Core Features:

- User Role Management: Admin, Teacher, and Student roles with distinct access permissions.
- Attendance Tracking: Log student attendance and use AI to predict no-shows based on past patterns, giving staff a tool to follow up in advance.
- Teacher Dashboard: A portal for teachers to add students and record attendance.
- Student Self-Check-In: Allow students to check in to their current class, based on class schedule
- Admin Management Tools: Tools for admins to add new teacher, students.
- Real-Time Reporting: Provide immediate snapshots of attendance to staff.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to inspire confidence and focus.
- Background color: Light cyan (#E0FFFF) for a calm, unobtrusive surface.
- Accent color: Dodger blue (#1E90FF) for active UI elements, with increased saturation.
- Body and headline font: 'PT Sans' (sans-serif) for its balance of modern form and readability.
- Use clear, recognizable icons from a standard set (e.g. Font Awesome) for key actions.
- Design a tiled dashboard layout optimized for desktop, with panels containing key reports, controls, and alerts. On smaller screens, stack the panels.
- Use subtle transitions for attendance logging, emphasizing responsiveness and feedback.
